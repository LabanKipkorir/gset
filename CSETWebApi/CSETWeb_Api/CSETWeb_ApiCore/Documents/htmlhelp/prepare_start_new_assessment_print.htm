CSET Gallery and Assessment Repository

 
After logging in, you have the option of choosing an existing assessment, importing an existing assessment, or starting a new assessment from the CSET Assessment Gallery.
 

Welcome to CSET

 
If you are new to CSET or do not have any assessments stored, you'll be routed to the Welcome to CSET page. From this page, you have the option to Start a New Assessment or Import an Existing Assessment.
 
 
To start a new assessment, users select the New Assessment tab.
 
The figure below shows the New Assessment Tab on the left side of the header. When selected, the New Assessment Tab reveals the CSET Assessment Gallery, followed by the My Assessments tab which shows any existing assessments.
 
 
 
After the user has created an assessment in CSET, the assessment will be stored (using the name chosen by the user) in the My Assessments repository.
 
 
 

New Assessment Button

Clicking the New Assessment button will start the assessment preparation process that will allow the user to address important areas before they can begin answering questions.
The first screen of the assessment preparation process is the Assessment Configuration screen.
