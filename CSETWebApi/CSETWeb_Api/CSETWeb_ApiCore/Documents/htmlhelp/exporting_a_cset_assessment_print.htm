Exporting a CSET Assessment

 
To export an assessment, simply select the Export button next to the assessment to be exported on the Landing page.
 

Export Button

 
Figure: Export button
 
To export an assessment file, click "Export" to the right of the assessment you wish to export. The downloaded .csetw file will be stored in your .local computer's Downloads folder, unless otherwise specified in your user settings.
