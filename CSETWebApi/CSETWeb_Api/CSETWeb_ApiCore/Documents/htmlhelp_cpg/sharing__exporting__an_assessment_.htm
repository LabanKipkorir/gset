4.3.1. Sharing (Exporting) an Assessment

 
To share an assessment with CISA or another internal or external stakeholder, CSET offers the following three (3) export functions:
 

1. Landing Page: Export as a .csetw file:

 
To export an assessment, simply select the Export button next to the assessment to be exported on the Landing page.
 

Export Button

 
Figure: Export button
 
To export an assessment file, click "Export" to the right of the assessment you wish to export. The downloaded .csetw file will be stored in your local computer's Downloads folder, unless otherwise specified in your user settings.
 

2. Tools Menu: Export in Excel (recommended for sharing basic data only, not recommended for full sharing functionality)

 
Selecting the "Export to Excel" link will download an Excel copy of your assessment results.
 
NOTE: Depending on assessment type underway by the user, the Excel report will be downloaded in the mode of the assessment taken, i.e. Questions vs. Requirements mode.
